  13. <img>

    14. Se utiliza para incrustar una imagen en una pagina HTML. Tecnicamente, las imagenes no se insertan en una pagina web; las imagenes estan vinculadas a paginas web. L etiqueta <ima> rea un espacio de espera para la imagen de referencia. Tiene atributos obligatorios:

    • src: especifica la ruta a la imagen
    • alt: especifica un texto alternativo para la imagen, si la imagen por algun motivo no se puede mostrar
    Ejemplo img
  14. <map>

    15. Se utiliza para definir un mapa de inagen, es una imagen con areas en las que se puede hacer clic. El atributo de nombre requerido del <map> esta asociado con el atributo usemap de <img> y crea una relacion entre la imagen y el mapa.

    Ejemplo map
